## Design Rules

### Safe Zones

- **Critical content** in central 70-80% of canvas
- **Text** avoid edges by at least 10% of width/height
- **CTA** bottom-right quadrant, clear of overlays

### Typography

| Rule | Specification |
|------|---------------|
| Max fonts | 2 typefaces |
| Min body size | 16px |
| Min headline | 32px |
| Line height | 1.2-1.5 for headlines |

### CTA Guidelines

| Element | Specification |
|---------|---------------|
| Position | Bottom-right quadrant |
| Height | Minimum 44px |
| Text | Action verb (Get, Start, Learn) |
| Contrast | 4.5:1 minimum |

### Text Coverage (Ads)

- Keep text under 20% of image area
- Meta penalizes heavy text in ads
- Use overlay panels for more text

### Print Specifications

| Requirement | Specification |
|-------------|---------------|
| Resolution | 300 DPI |
| Color Mode | CMYK |
| Bleed | 3-5mm |
| Safe margin | 5-10mm from trim |

## Design Workflow

### Step 1: Gather Requirements

1. **Purpose** - Social cover, ad banner, website hero, print?
2. **Platform** - Which platform or custom dimensions?
3. **Content** - Headline, subtext, CTA, logo?
4. **Brand** - Existing brand guidelines?
5. **Style** - Art direction preference?
6. **Quantity** - How many options?

### Step 2: Research & Inspiration

1. Review brand guidelines
2. Research competitor banners
3. Browse design references (Pinterest, Dribbble)
4. Select 2-3 complementary styles

### Step 3: Design Options

For each art direction:

1. Create base composition
2. Apply brand colors and typography
3. Add visual elements (photos, graphics)
4. Position text and CTA
5. Verify safe zones and contrast

### Step 4: Export

| Format | Use Case |
|--------|----------|
| PNG | Web, social media |
| JPG | Photos, smaller file size |
| SVG | Logos, simple graphics |
| PDF | Print production |

## Quality Checklist

### Before Export

- [ ] Correct dimensions for platform
- [ ] Safe zones respected
- [ ] Text contrast 4.5:1 minimum
- [ ] CTA visible and clickable
- [ ] Logo properly placed
- [ ] Brand colors applied
- [ ] No pixelation or blur
- [ ] File size optimized

### For Ads

- [ ] Text under 20% coverage
- [ ] CTA clear and actionable
- [ ] Value proposition visible
- [ ] Complies with platform guidelines

### For Print

- [ ] 300 DPI resolution
- [ ] CMYK color mode
- [ ] Bleed included
- [ ] Spell check complete
