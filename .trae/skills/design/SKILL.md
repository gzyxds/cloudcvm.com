# Design

Unified design skill for brand identity, visual design, and creative assets.

## When to Use

- Brand identity, voice, and visual assets
- Logo design and variations
- Banner design for social media, ads, web, print
- Icon design and icon sets
- Presentations and pitch decks
- Social media graphics
- Corporate identity program (CIP) deliverables

## Logo Design

### Logo Design Principles

1. **Simplicity** - Simple logos are more memorable and versatile
2. **Scalability** - Must work at 16px and 1000px+
3. **Timelessness** - Avoid trendy elements that date quickly
4. **Versatility** - Work in color, black, and white
5. **Relevance** - Reflect the brand's industry and values

### Logo Styles

| Style | Characteristics | Best For |
|-------|-----------------|----------|
| Minimalist | Clean lines, simple shapes | Tech, SaaS, startups |
| Wordmark | Typography-focused | Brands with unique names |
| Lettermark | Initials/acronyms | Long company names |
| Pictorial | Icon + text | Consumer brands |
| Abstract | Geometric shapes | Tech, innovation |
| Vintage | Retro elements, textures | Craft, artisan, heritage |
| Badge/Emblem | Contained design | Food, beverage, automotive |
| Mascot | Character-based | Sports, food, family brands |

### Logo Color Psychology

| Color | Emotions | Industries |
|-------|----------|------------|
| Blue | Trust, stability | Finance, tech, healthcare |
| Red | Energy, passion | Food, entertainment, retail |
| Green | Growth, nature | Health, environment, finance |
| Yellow | Optimism, clarity | Food, children, creative |
| Purple | Luxury, creativity | Beauty, luxury, tech |
| Orange | Energy, warmth | Food, sports, youth brands |
| Black | Sophistication, power | Luxury, fashion, tech |
| White | Purity, simplicity | Healthcare, minimal brands |

### Logo Design Checklist

- [ ] Works in single color (black/white)
- [ ] Legible at small sizes (favicon size)
- [ ] Has adequate whitespace/clear space
- [ ] Vector format (SVG, AI, EPS)
- [ ] Multiple file formats provided
- [ ] Consistent across applications

### Banner Design Rules

1. **Safe Zones** - Keep critical content in central 70-80%
2. **One CTA** - Single call-to-action, bottom-right, min 44px height
3. **Typography** - Max 2 fonts, min 16px body, >=32px headline
4. **Text Coverage** - Keep text under 20% for ads (Meta penalizes)
5. **Print Specs** - 300 DPI, CMYK, 3-5mm bleed

## Icon Design

### Icon Styles

| Style | Characteristics | Best For |
|-------|-----------------|----------|
| Outlined/Line | Stroke-based, minimal | UI interfaces, web apps |
| Filled/Solid | Solid shapes | Mobile apps, nav bars |
| Duotone | Two-color, layered | Marketing, landing pages |
| Rounded | Soft corners | Friendly apps, health |
| Sharp | Angular corners | Tech, fintech, enterprise |
| Flat | No shadows, simple | Material design, Google-style |
| Gradient | Color gradients | Modern brands, SaaS |
| 3D | Depth and shadows | Gaming, immersive apps |

### Icon Design Guidelines

1. **Consistent Stroke Width** - Use same stroke across icon set
2. **Optical Alignment** - Visually center, not mathematically
3. **Pixel-Perfect** - Align to pixel grid for crisp rendering
4. **Size Consistency** - Maintain visual weight across icons
5. **Accessibility** - Ensure sufficient contrast (4.5:1 minimum)

### Common Icon Sizes

| Size | Use Case |

| 20px | Small UI elements |
| 24px | Standard UI icons |
| 32px | Navigation, toolbars |

| 64px | App icons, large features |

## Presentation Design

### Slide Structure

1. **Title Slide** - Brand, title, presenter
2. **Problem/Context** - Set the stage
3. **Solution** - Your proposition
4. **Features/Benefits** - Key points
5. **Evidence** - Data, testimonials, case studies
6. **Call to Action** - Next steps
7. **Contact/Q&A** - Closing

### Slide Design Principles

1. **One Idea Per Slide** - Don't overcrowd
2. **Visual Hierarchy** - Title > Key Point > Details
3. **Consistent Layout** - Grid-based alignment
4. **Limited Text** - 6 words per line, 6 lines max
5. **High-Quality Images** - No watermarks, no pixelation
6. **Brand Consistency** - Colors, fonts, logo placement

### Typography in Presentations

| Element | Size (for 1920x1080) |
|---------|---------------------|
| Title | 44-60pt |
| Subtitle | 24-32pt |
| Body | 18-24pt |
| Captions | 14-16pt |

## Social Media Graphics

### Platform-Specific Guidelines

#### Instagram
- Square posts (1080x1080) for feed
- Portrait (1080x1350) for more vertical space
- Stories/Reels (1080x1920) full screen
- Keep text minimal, focus on visuals

#### LinkedIn
- Professional tone
- 1200x627 for posts
- Less text-heavy than Instagram
- Data and statistics perform well

#### Twitter/X
- 1200x675 for image posts
- Text overlay works well
- Memes and GIFs for engagement

#### Facebook
- 1200x630 for link previews
- 1200x1200 for square posts
- Video gets more reach

### Social Media Design Tips

1. **Mobile-First** - Most users view on mobile
2. **Text Overlay** - Readable without sound (for video)
3. **Brand Consistency** - Recognizable across platforms
4. **CTA Clarity** - Clear next step
5. **Accessibility** - Alt text, high contrast

## Corporate Identity Program (CIP)

### Common CIP Deliverables

| Category | Items |
|----------|-------|
| Stationery | Business cards, letterhead, envelopes |
| Marketing | Brochures, flyers, posters |
| Digital | Email signatures, social templates |
| Signage | Office signs, vehicle wraps |
| Apparel | Uniforms, promotional items |
| Packaging | Product packaging, labels |

### CIP Design Process

1. **Define Brand Guidelines** - Colors, fonts, logo usage
2. **Create Templates** - Consistent starting points
3. **Design Core Items** - Business cards, letterhead first
4. **Expand to Marketing** - Brochures, presentations
5. **Document Usage** - Guidelines for each item

## Design Workflow

### New Brand Project

1. **Research** - Industry, competitors, audience
2. **Strategy** - Positioning, personality, values
3. **Visual Identity** - Logo, colors, typography
4. **Brand Guidelines** - Usage rules, examples
5. **Applications** - Stationery, digital, marketing

### Design Review Checklist

- [ ] Aligns with brand guidelines
- [ ] Appropriate for target audience
- [ ] Works across required formats
- [ ] Accessible (contrast, legibility)
- [ ] Technically correct (resolution, color mode)
- [ ] Consistent with other brand materials
