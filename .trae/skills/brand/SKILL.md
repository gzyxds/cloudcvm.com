# Brand

Brand identity, voice, messaging, and visual identity standards.

## When to Use

- Brand voice definition and content tone guidance
- Visual identity standards and style guide development
- Messaging framework creation
- Brand consistency review and audit
- Asset organization, naming, and approval
- Color palette management and typography specs

## Brand Identity Components

### Brand Voice

Brand voice defines how your brand communicates. It should be:

1. **Consistent** - Same personality across all channels
2. **Distinctive** - Different from competitors
3. **Authentic** - True to your values and mission
4. **Appropriate** - Suitable for your audience

#### Voice Attributes

| Attribute | Description | Example |
|-----------|-------------|---------|
| Professional | Formal, expert tone | Enterprise software |
| Friendly | Warm, approachable | Consumer apps |
| Bold | Confident, assertive | Sports, gaming |
| Playful | Fun, lighthearted | Children's products |
| Luxurious | Sophisticated, exclusive | High-end brands |
| Technical | Detailed, precise | Developer tools |

### Visual Identity

#### Logo Guidelines

1. **Clear Space** - Minimum whitespace around logo
2. **Minimum Size** - Smallest acceptable reproduction
3. **Color Variants** - Full color, monochrome, reversed
4. **Incorrect Usage** - Don'ts for logo application
5. **File Formats** - Vector (SVG, AI) and raster (PNG)

#### Color Palette

| Category | Purpose | Examples |
|----------|---------|----------|
| Primary | Main brand color | Logo, CTAs, key elements |
| Secondary | Supporting colors | Accents, highlights |
| Neutral | Backgrounds, text | White, grays, black |
| Semantic | Functional meaning | Success (green), Error (red) |

#### Typography

| Element | Usage | Guidelines |
|---------|-------|------------|
| Display | Headlines, hero text | Bold, distinctive |
| Heading | Section titles | Clear hierarchy |
| Body | Paragraphs, content | Readable, comfortable |
| Mono | Code, data | Technical content |

### Messaging Framework

#### Brand Positioning Statement

```
For [target audience],
[Brand name] is the [category]
that [key benefit]
because [reason to believe].
```

#### Value Proposition

1. **What you do** - Core offering
2. **Who you serve** - Target customer
3. **How you're different** - Unique advantage
4. **Why it matters** - Customer benefit

#### Key Messages

| Audience | Message | Proof Point |
|----------|---------|-------------|
| Primary | Main message | Evidence |
| Secondary | Supporting message | Evidence |

## Brand Consistency Checklist

### Visual Elements
- [ ] Logo used correctly with proper clear space
- [ ] Colors match brand palette
- [ ] Typography follows brand guidelines
- [ ] Imagery aligns with brand style
- [ ] Layout follows brand templates

### Voice & Messaging
- [ ] Tone matches brand personality
- [ ] Key messages are consistent
- [ ] Vocabulary aligns with brand language
- [ ] Content serves target audience

### Asset Management
- [ ] Files named consistently
- [ ] Correct file formats used
- [ ] Versions tracked and organized
- [ ] Outdated assets archived

## Color Palette Management

### Color Naming Convention

```
color-{semantic}-{variant}

Examples:
color-primary-500
color-secondary-300
color-neutral-900
color-success-500
```

### Color Accessibility

| Ratio | Use Case |
|-------|----------|
| 4.5:1 | Normal text (AA) |
| 3:1 | Large text (AA) |
| 7:1 | Normal text (AAA) |
| 4.5:1 | UI components (AA) |

### Color Psychology

| Color | Associations | Industries |
|-------|--------------|------------|
| Blue | Trust, stability | Finance, tech, healthcare |
| Red | Energy, urgency | Food, entertainment, retail |
| Green | Growth, nature | Health, environment, finance |
| Yellow | Optimism, clarity | Food, children, creative |
| Purple | Luxury, creativity | Beauty, luxury, tech |
| Orange | Energy, warmth | Food, sports, youth brands |
| Black | Sophistication | Luxury, fashion, tech |

## Typography Specifications

### Font Selection Criteria

1. **Legibility** - Clear at all sizes
2. **Character** - Matches brand personality
3. **Versatility** - Works across applications
4. **Availability** - Web-safe or licensed

### Type Scale

| Name | Size | Use |
|------|------|-----|
| xs | 12px | Captions, footnotes |
| sm | 14px | Small text, labels |
| base | 16px | Body text |
| lg | 18px | Large body |
| xl | 20px | Subheadings |
| 2xl | 24px | Headings |
| 3xl | 30px | Large headings |
| 4xl | 36px | Display text |
| 5xl | 48px | Hero text |

## Asset Organization

### File Naming Convention

```
{brand}-{type}-{variant}-{size}.{ext}

Examples:
acme-logo-primary-fullcolor.svg
acme-logo-primary-monochrome.svg
acme-icon-home-24.svg
acme-photo-hero-1920x1080.jpg
```
